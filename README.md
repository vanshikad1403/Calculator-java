# Simple Calculator in Java  
A basic Java console-based calculator that performs arithmetic operations using clean code and proper error handling.

## Features  
- Addition, subtraction, multiplication, and division  
- Handles invalid inputs using exceptions  
- Prevents division by zero  
- Menu-driven structure for easy user interaction  

## Concepts Used  
- Object-Oriented Programming (OOP)  
- Methods for individual operations  
- Exception handling (try-catch)  
- Input validation using Scanner  

## How It Works  
The user selects an operation from the menu, enters two numbers, and the calculator performs the chosen operation while ensuring safe execution through exception handling.

## Technologies  
- Java  
- Exception Handling  
- Scanner API
